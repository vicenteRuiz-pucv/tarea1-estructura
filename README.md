## Código de Ejemplo (tarea1)
Para ejecutar el ejemplo tarea1.c primero debemos guardar los cambios!
````
gcc tdas/*.c tarea1.c -o tarea1
````

Y luego ejecutar la version actualizada:
````
./tarea1
````
